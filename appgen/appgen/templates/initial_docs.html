Welcome to the Madrona Virtual Machine

To get started please minimize the virtual machine window and

  1. Add the following to your hosts file

    Windows

    • Navigate to C:\Windows\System32\drivers\etc\
    • Open the hosts file for editing (may require admin privledges)
    • Add the following line to the end of the file and save.
      {{ip}} {{domain}}

  2. Visit {{active_app_url}} in your host web browser
