Welcome to the Madrona Virtual Machine

To get started, we'll need to set up your system to recognize the virutal machine by the domain name madrona.

  1. Set up your hosts file to point to the IP address of the virtual machine

    Windows

    • Navigate to C:\Windows\System32\drivers\etc\
    • Open the hosts file for editing (may require admin privledges)
    • Add the following line to the end of the file and save.
      {{ip}} {{domain}}

    Mac OSX or Linux

    • Open the /etc/hosts file for editing (may require root privledges)
    • Add the following line to the end of the file and save.
      {{ip}} {{domain}}

    Note: you'll need to revisit step #1 every time the VM is assigned a new IP address.
    Use the ifconfig command to check or reload this page.

  2. Visit {{appgen_url}} in your web browser to begin.

    Note: There are several accounts pre-set on this machine (including the unix, database and django account),
    All of which have a username & password of "madrona" & "madrona".
